# AI Resume Analyzer

AI Resume Analyzer helps job seekers understand how well their resume matches a target role. Upload a resume and provide a job description to receive an ATS-style score, skill and keyword comparisons, section feedback, and practical improvement suggestions.

## Access the app

Use the live application at [resume-analyzer-sage-pi.vercel.app](https://resume-analyzer-sage-pi.vercel.app/).

## How to use it

1. Open the live application.
2. Upload your resume as a PDF or DOCX file.
3. Paste the job description, or upload it as a PDF or DOCX file.
4. Select **Analyze Resume**.
5. Review your score, skills, keywords, section analysis, recommendations, and improvement roadmap.
6. Select **Report** to download a copy of the analysis.
